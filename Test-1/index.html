Quale è la differenza tra i ruoli di HTML, CSS e Javascript?

Usiamo il linguaggio di markup html per indiacare i principali elementi della nostra pagina web.
Utilizziamo CSS per migliorare l'aspetto estetico della nostra pagina;
Ed infine utilizziamo javascript per collegare alcuni elementi della nostra pagina ad un eventuale database.


Quali sono i tag HTML di base?

I principali tag sono:

  • Tag: div
  • Tag: h1-h6
  • Tag: ul
  • Tag: ol
  • Tag: section
  • Tag: img

Quali sono i principali elementi Javascript che conosci?

I principali elementi Javascript che conosco sono:

  • Dichiarazione variabili con 'var' e 'let'
  • Istruzione di controllo 'if', 'else-if'
  • Ciclo 'for'
  • Lunghezza array 'array.length'
  • Stampare output con 'console.log()'

Quale è la differenza tra una tipizzazzione debole e una forte?

  • Tipizzazione debole: le variabili fanno riferimento a valori di qualisi tipo (var, let).
  • Tipizzazione forte: le variabili fanno riferimento a valori ben definiti (int, char, float, string, boolean).

Qual'è il punto di collegamento tra front-end e back-and?

Il punto di collegamento è dato dal linguaggio di programmazione che viene, utilizzato. Ci sono diversi linguaggi che permettono il collegamento tra front-end e back-end come Java, PHP e python.
